# Caption Genie

AI-powered caption generator for creators and businesses.

## ✨ Features
- Instagram captions  
- TikTok captions  
- Facebook captions  
- LinkedIn captions  
- X (Twitter) captions  
- YouTube captions  
- Multiple tone options  
- Personal Story integration  
- Groq AI powered  

## 🛠 Tech Stack
- Replit  
- Groq API  
- Llama 3.3 70B  
- React
